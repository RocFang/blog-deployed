为什么这两个月没更新博客

2019-11-03
#博客

最近这两个月，我没有在本博客上更新一篇文章。这也不是什么罕见的事情，因为各种借口不愿意写文章在过去也发生过很多次。

但这次还是有一些不一样。我其实还是在写文章，尤其是技术相关类的文章，写了不少。但如在重新打造我的笔记系统中提到的，我现在在用 github 的 issues 来构建我的笔记系统，在写那篇文章时，只是刚刚将一种想法付诸行动，而这两个月，我开始迷上了这种记录的方式，自然就在博客上花的时间少了。

博客我也是不愿意荒废的，以后我会尽量平衡好这两者的关系，一些适合公开的技术笔记也会放出来。
